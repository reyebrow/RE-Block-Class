=====
RE Block Class
-----
RE Block Class was developed by Raised Eyebrow Web Studio <http://raisedeyebrow.com>.

It was created to assign classes belonging to the Foundation framework <http://foundation.zurb.com> to individual blocks.

CHECK OUT A BRANCH TO SEE CODE

=====
Installation
-----

1. Enable the module
2. To add a class to a block, simply visit that block's configuration page at Administration > Structure > Blocks
